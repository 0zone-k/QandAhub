# アプリケーションを作成した背景
前職で、業務内容や商品内容について本社・支社への質問しずらいという課題があった。
課題を分析したところ、営業所勤務の社員は出入りが激しく、また、営業職員の対応に追われなかなか内線をかけるタイミングがない、掛けても担当者が不在、担当者がわからないということが原因だと仮定した。
同様の問題を抱えている方も多いと推測し、真因を解決するために、部署間でのQ＆Aをすることができるアプリケーションを開発した。
